# Site Lubeemel

Atividade sugerida em aula com o objetivo de estudar frameworks e links.
Ela consistia em escolher um produto e criar o site de uma empresa fictícia explorando as ferramentas do framework, além de conter o link do site de todos os colegas de classe.
No caso deste projeto, o produto escolhido foi Mel, e a empresa fictícia recebeu o nome Lubeemel.
